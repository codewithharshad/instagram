# ToDo App

A simple ToDo app that allows users to manage their tasks. Tasks are stored locally in the browser's storage.

## Features

- Add new tasks
- Mark tasks as completed
- Remove tasks
- Tasks are stored in the browser's local storage

## Technologies Used

- HTML
- CSS
- JavaScript
